Mycelial Network Simulation

This simulation allows you to explore the growth patterns and behaviors of mycelial networks. The controls are organized into categories to help you focus on specific aspects of the simulation.

Interface Categories

Visual - Controls the appearance of the mycelium
Biology - Biological properties of the mycelial network
Growth - Controls growth patterns and branching
Environment - Environmental factors and nutrients
Advanced - Advanced biological parameters

Presets

Try different presets to see various growth patterns:

  • Default - Balanced settings for typical growth
  • Dense Growth - Creates a dense, heavily branched network
  • Minimal - Sparse, minimalist growth pattern
  • Colorful - Vibrant purple coloration for artistic effect
  • Realistic - More biologically realistic growth behaviors

Save and Load

You can save your custom settings and load them later using the Preset Management folder.

Key Parameters

Main Branch Count - Number of primary hyphae
Branch Chance - Probability of branching
Chemotropism - How strongly hyphae grow toward nutrients
Nutrient Distribution - Controls resource availability
